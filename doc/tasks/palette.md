## 任务清单

- [ ] **1. 定义 NES 64 色系统调色板**
  - 文件：`palette.py`
  - 定义 `SYSTEM_PALETTE` 列表：64 个元组，每个 `(R, G, B)` 0–255
  - 完整的 64 色 NES 标准调色板（0x00–0x3F）
  - 参考：https://www.nesdev.org/wiki/PPU_palettes
  - 预计代码量：~20 行（64 行数据）

- [ ] **2. 编写数据完整性校验测试**
  - 文件：`tests/test_palette.py`
  - 测试用例：
    - 长度为 64
    - 每个元素为 (R, G, B) 三元组
    - R/G/B 均在 0–255 范围
    - 部分已知颜色校验（如 0x0F = 黑色 (0,0,0)，0x30 = 白色 (255,255,255)）
  - 预计代码量：~15 行

- [ ] **3. 跑通测试**
  - `python -m pytest tests/test_palette.py -v`
